Quick Start
1. Open an image — press o or drag & drop a file
2. Preview the result — original and processed side by side
3. Tweak settings — switch to the Settings tab to adjust
4. Save — press s to export the cleaned-up image
What pixfix does
Cleans up AI-generated pixel art by snapping pixels to a clean grid, removing anti-aliasing fuzz, reducing to a chosen color palette, and removing backgrounds.
